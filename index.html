번개유심이란?

  • 번개유심은 SK텔링크와 번개장터의 제휴상품으로서, SK텔링크가 후불 유심 이동전화 서비스를, 번개장터가 번개포인트를 제공합니다.
  • 고객은 상품을 이용하기 위해 SK텔링크의 알뜰폰 브랜드인 SK 7mobile에서 번개유심 이동전화상품을 가입/개통하여야 하며, 가입/개통한 전화번호로 번개장터에 회원가입을 함으로써 번개포인트를 수령/이용하실 수 있습니다.

번개포인트 사용안내

  • 번개장터에서 거래시 번개페이 이용하여 포인트를 현금처럼 사용 가능 (단, 번개포인트는 배송비로 사용 불가)
  • 100원 단위로 사용 가능
  • 1회 5만원까지 사용 가능
  • 포인트 차감시 최소상품 금액은 1,000원 이상시 사용 가능
  • 번개포인트(웰컴포인트)의 유효기간은 3개월

번개포인트 제공방법

  • 모든 번개유심 요금제는 가입 시 번개포인트(웰컴포인트)를 1회 제공함
  • 웰컴포인트는 개통 시 요금제를 기준으로 제공되며 개통 익월 17일 전 후에 지급함
  • 웰컴포인트는 지급시점에 일시정지, 명의변경, 해지, 번개요금제 외 타요금제로의 변경 등이 없는 가입고객에게만 제공함
  • 꼭 확인하세요!
    번개포인트를 제공받기 위해서는 “번개유심 요금제로 가입된 전화 번호로 반드시 번개장터에 회원가입을 하였거나 새로 하여야 함”
  • 포인트를 받지 못하는 경우가 발생하지 않도록 번개유심 요금제 개통 완료 후 반드시 “번개장터 애플리케이션 또는 홈페이지에서 해당 전화 번호로 회원가입”을 해주시기 바랍니다.
  • 기존에 “해당 전화번호로 이미 번개장터에 회원가입을 하신 분”은 별도로 다시 회원가입하실 필요가 없습니다.

유의 사항

  • 본 이벤트를 포함하여 유심 요금제는 신규가입(새로운 번호)/ 번호이동 (쓰던 번호 그대로 통신사만 변경)으로만 가입이 가능하며, 기존 가입 고객님의 단순요금제 변경은 불가합니다.
  • 현재 SKT를 이용중이신 고객님들도 ‘번호이동’으로 가입하셔야 하며, 번호 이동 시 기존 통신사의 위약금 사전 확인을 권장 드립니다. (번호이동 시 기존 통신사는 자동 해지)
  • 할인 혜택은 해지 전까지 평생 적용되며, 가입 첫 달은 일할 할인 계산 됩니다. (할인 금액이 포함된 요금은 명세서에서 확인이 가능 실시간 요금조회에서 확인 불가)
  • 이용 중 명의변경, 해지, 개통 월 중 다른 요금제로 변경 시에는 할인혜택이 중지 또는 제외됩니다. (일할 할인 계산되지 않음)
  • 일시 정지가 포함된 월에는 요금할인이 적용되지 않습니다.
    단, 일시정지가 끝난 다음 달부터는 동일하게 할인 혜택이 제공됩니다.
